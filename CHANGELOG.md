## [1.0.0] — 2026-04

### 全新特性

- 全新 UI 大改版 — Tailwind CSS v4 重构全部界面，支持三套主题（Steam Dark / Midnight / Classic）
- 完整双语支持 — 中文 / 英文 / 跟随系统语言自动切换
- 评测模式 — 在 Steam 游戏商店页直接创建和更新评测，支持推荐/不推荐、可见性、语言等设置
- 游戏截图浏览 — 图片池新增「截图」标签页，可浏览并拖入游戏截图
- 外链图片插入 — 右键菜单支持插入外部 HTTPS 图片链接，自动检测 Steam 兼容性
- 图片上传优化 — Steam 2MB 大小校验、批量重命名、拖入路由修正
- 新手引导系统 — 基础 + 高级双阶段交互式引导，随时可从设置中重播
- 统一错误处理 — Steam EResult 错误码分类 + 可读的 i18n 错误消息
- Toast 通知系统 — 替换所有 window.alert()，操作反馈更优雅

### 改进

- 工具栏三种停靠模式 — 侧边、顶部、浮动，自由切换
- 骨架屏 + 过渡动画 — 编辑器、图片池、预览面板、弹窗等全面加入加载与过渡动效
- 章节导航双模式 — 固定模式（嵌入侧边栏）和可拖拽模式（自由浮动）
- 多窗口 session 隔离 — 同时打开多个编辑器窗口不再互相干扰
- 嵌套表格支持 — 右键菜单支持在表格单元格内插入子表格
- 右键菜单防溢出 — 菜单渲染后自动调整位置，不超出视口
- Dialog 系统增强 — 支持多行文本输入、字数显示

### 修复

- BBCode 往返修复 — [code] 内容不再被解析、[url] 无等号格式、inline+sizeFull 图片标签、文件名后缀污染
- 图片池修复 — 粘贴多图只处理第一张、缩略图 URL 误用导致图片模糊
- 多 Steam 标签页路由修复 — 多个 Steam 页面同时打开时优先路由到指南编辑页
- 草稿隔离修复 — activeDraftId 多窗口隔离、评测草稿绑定游戏 appId 防止跨游戏覆盖
- 预览面板竞态修复 — guideInfo 竞态条件 + 章节切换骨架屏时机

### 移除

- 移除「草稿绑定」功能 — 该功能设计不合理，已废弃。草稿现在按模式自动隔离，无需手动绑定
